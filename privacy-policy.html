Privacy Policy

Last Updated: [18 November 2025]

This Privacy Policy explains how our Chrome extension (“Extension”) handles user data while providing AI-powered website summarization, link extraction, and query-based assistance.

1. Information We Collect

The Extension collects the following information only when a user actively interacts with it:

  • Current tab URL – The URL of the website the user has opened.
  • Website content & links – The Extension fetches the visible content and hyperlinks from the webpage for the sole purpose of generating answers.
  • User queries – The questions the user types into the Extension.

No other personal data is collected.

2. How We Use the Information

The collected data is used strictly to provide the features of the Extension, including:

  • Processing the current webpage and extracting its content
  • Analyzing links found on the page
  • Answering user questions related to the website
  • Displaying reference links from the processed content

The data is temporarily sent to our server at https://gemini-react-chatbot.onrender.com to generate responses using Google Gemini AI.

3. No Data Storage

We do not store any URLs, website content, user queries, or personal data on our servers or databases. All processing is done temporarily and discarded immediately after generating the answer.

4. No Selling or Sharing of Data

We do not sell, share, or transfer user data to any third parties. We do not use user data for advertising, marketing, or analytics.

5. Permissions Explained

The Extension may require the following Chrome permissions:

  • activeTab – Used only to read the URL of the page the user is currently viewing.
  • storage (if applicable) – To store temporary session IDs during usage.
  • scripting / host permissions – Only to read content from the active website for processing.

These permissions are used solely for the core functionality of the Extension.

6. Security

All data exchanged between the Extension and our backend API is transmitted using HTTPS encryption to protect user privacy.

7. Third-Party Services

Our backend is hosted on Render, and we use Google Gemini API for AI processing.
Google's Privacy Policy: https://policies.google.com/privacy

8. Children’s Privacy

This Extension is not intended for users under 13 years of age.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be reflected with a new “Last Updated” date.
